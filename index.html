API Diccionario - Asier Luque Ventura

Esta aplicación web utiliza una API de diccionario gratuito, para ver un ejemplo del objeto que devuelve pulse aquí.
Para su correcto funcionamiento, hay que introducir una palabra en inglés, un ejemplo con contenido completo es la palabra "what", aunque cualquier palabra es válida. Busque la palabra "pal" para un ejemplo con dos resultados.
